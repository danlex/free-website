Need the full version?

Here is the detailed flow from clone to live site.

1

Clone the repo to inspect the system

Start by reviewing the explainer, the starter folder, and the quickstart so you know exactly which files belong to your own site. 

git clone https://github.com/danlex/codex-free-website.git
cd codex-free-website
2

Use the starter folder

Copy the files from templates/no-framework into your own repo or working directory. That is the starting point for your website. 

templates/no-framework/
  index.html
styles.css
script.js
3

Open it in Codex and customize

Rewrite the copy, change the sections, update the links, and make the design feel specific to the person using it.

Good prompt: “Turn this starter into a personal website for [name], a [role]. Update the hero, about section, projects, links, and SEO copy. Keep it compatible with GitHub Pages.”

4

Push to GitHub

Create a new repo with the starter files, commit them, and push main. 

git init
git add .
git commit -m "Initial website starter"
git branch -M main
git remote add origin https://github.com/your-username/your-repo-name.git
git push -u origin main
5

Enable GitHub Pages

Your own starter site can publish directly from the root branch with no build step.

  • Open Settings → Pages
  • Choose Deploy from a branch
  • Select main and / (root)
6

Connect the custom domain

Rename CNAME.example to CNAME, put the domain in the file, update DNS, then enable HTTPS in GitHub Pages.

Domain setup

The last 10 percent that makes it feel real.

Subdomain

For www.yourdomain.com, add a CNAME record pointing www to your-username.github.io.

Apex domain

For yourdomain.com, add GitHub Pages A records and, if you want, AAAA records for IPv6.

GitHub setup

Add the custom domain in Pages settings and enable HTTPS once DNS has propagated.

Publish checklist

Before you share the repo.

Replace placeholder copy and sections

Add real links and contact info
Rename CNAME.example when needed
Set the domain inside the CNAME file
Enable GitHub Pages and HTTPS
